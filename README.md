# Mikasa-AI-Version-1.0

##  Meet Mikasa - Your Personalized AI Assistant

Hey there! I'm **Charan Konchada**, an AI enthusiast on a mission to build an AI model that works like **ChatGPT**, but with a twist—**Mikasa** remembers your preferences and can be fully personalized based on how *you* want to interact. 💡🤖

###  What is Mikasa?

Mikasa is an AI assistant powered by a local LLM (Large Language Model), designed to:
- Remember your preferences
- Adapt its personality
- Be more than just a chatbot—it can be your **best friend**, **brother**, **sister**, or anything you imagine 💬❤️

---

###  Personality Modes

- **Assistant Mode** (default): A professional, helpful assistant.
- **Mikasa Mode** (custom): Fully customizable! Choose how Mikasa talks to you—casual, caring, teasing, or anything you like. You can even rename this mode in the code to match your vibe.

---

###  Prerequisites

Before running the code:

1. **Download & Install [Ollama](https://ollama.com/)** – this is required to run local LLMs.
2. After installation, open your **Command Prompt** or **Terminal** and run:

   ```
   ollama run openchat:7b
   ```

   This will download the required **OpenChat:7B** model.

3. Once the model is ready, you're all set to run Mikasa!

---

###  Precautions

- Make sure you have a **stable internet connection** while downloading the model—it may take time depending on your speed.
- **Use a system with sufficient RAM & storage** (recommended: 8GB RAM or more) to avoid performance issues.
- This model runs **locally**, so it might consume CPU/GPU resources—don’t worry if your system fans get loud!
- Always **check the code and dependencies** before running. Safety first.
